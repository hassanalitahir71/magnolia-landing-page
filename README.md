# Magnolia Travels Landing Page ✈️🌍

A modern, fully responsive landing page for "Magnolia Travels," a fictional travel agency. Built using React and Tailwind CSS, this project features a clean UI, interactive hover states, and a mobile-friendly design to help users browse destinations, view services, and book trips.

## 📸 Previews



## ✨ Features

* **Responsive Navigation:** Fully functional mobile menu and desktop navbar.
* **Hero Section with Search Widget:** A visually appealing hero area with a simulated flight/hotel booking search UI.
* **Service & Destination Cards:** Grid layouts showcasing travel packages and company services with modern hover effects.
* **Testimonials Carousel:** Interactive slider built with `react-slick` to display customer reviews.
* **Newsletter Subscription:** A clean UI component for capturing user emails.
* **Comprehensive Footer:** Includes social links, contact information, and app store download buttons.

## 🛠️ Tech Stack

* **Framework:** [React](https://reactjs.org/) (Vite or Create React App)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** [React Icons](https://react-icons.github.io/react-icons/)
* **Carousel:** [React Slick](https://react-slick.neostack.com/)

## 📂 Project Structure

The project is broken down into modular React components for easy maintenance:

* `Header.jsx` - Top navigation bar.
* `Herosection.jsx` - Main banner and search widget.
* `OurServices.jsx` - Grid displaying company perks.
* `DestinationSection.jsx` - Travel package cards.
* `TestimonialsSection.jsx` - Customer review slider.
* `EmailSection.jsx` - Newsletter signup banner.
* `Footer.jsx` - Bottom site directory and contact info.
